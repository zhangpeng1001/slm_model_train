安装步骤：
    下载 Anaconda 安装包（推荐使用 清华镜像 加速下载）:https://mirrors.tuna.tsinghua.edu.cn/anaconda/archive/
    安装时选择自定义路径（如 D:\Anaconda3），取消勾选“Add Anaconda to PATH”。
    勾选“Register Anaconda as my default Python”（仅当希望 Anaconda 的 Python 成为默认环境）。
安装后配置：
    手动添加 Anaconda 的 Scripts 目录到系统 PATH。
    配置国内镜像源加速包下载（如清华源）：
    conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
    conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
    conda config --set show_channel_urls yes
    conda config --remove-key channels #如果镜像有问题，可以移除：移除清华镜像源
    conda config --remove channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main # 移除有问题的镜像源
使用建议：
    通过 Anaconda Navigator 图形界面管理环境和包（适合新手）。
    使用 conda 命令行工具灵活控制环境（适合高级用户）。

# 查看 conda 版本（验证 conda 安装）
conda --version
# 列出所有 conda 环境
conda env list
# 运行 Conda 初始化命令[命令执行完，需要重启命令窗口]
conda init


虚拟环境管理
使用虚拟环境隔离项目：
Anaconda 的核心优势是虚拟环境管理。建议为每个项目创建独立的虚拟环境，避免依赖冲突。
创建虚拟环境示例：
    conda create -n myenv python=3.9  # 创建名为 myenv 的环境，指定 Python 3.9
        conda create -n huggingface python=3.12
        conda create -n huggingface python=3.12 -c defaults  #可以在创建环境时临时指定使用 conda 官方源 -c defaults 表示强制使用 conda 官方默认源
        D:\worksoft\install\Anaconda\envs\huggingface #环境存放的位置
    conda activate myenv              # 激活环境
        [conda activate huggingface ]
    conda install numpy pandas      # 安装包到当前环境
        pip install requests        # Conda 仓库未覆盖的包再用 pip
    conda deactivate #退出环境
切换环境：
    使用 conda deactivate 退出当前环境，conda activate <env_name> 切换到其他环境。
